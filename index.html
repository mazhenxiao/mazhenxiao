iss前端框架

简介

iss.js是一个脚手架，用来自调用source文件夹下的calendar日历控件、dialog弹窗控件、scrollImageg滚图控件、unslider滚图控件、uploadify上传控件、validate验证控件等。严禁用户自己在页面中引用source文件夹下除iss.js以外的任何js控件。

在页面中引用iss框架:

<link type="text/css" rel="stylesheet" href="css/css.css"> -------------------TOPK通用css，可不加载自己针对项目重写
<link type="text/css" rel="stylesheet" href="css/iss.css"> --------------------控件必须css
<script type="text/javascript" src="Scripts/jquery-1.11.1.js"></script> --基于jquery开发
<script type="text/javascript" src="Scripts/source/iss.js"></script> ------框架基础js
(请关注iss.js变量： ISS_SROUCE_URL = "Scripts/source/"; 页面引入js资源地址配置，可更换)
如需esayui单独引入
<script type="text/javascript" src="Scripts/jquery.easyui.min.js"></script>
<link type="text/css" rel="stylesheet" href="css/easyui.css">

常用小工具

parseInput: 输入类型限制 $(xx).parseInput({})

1、 $("input").parseInput({type:'int',dec:2});
2、 $("input").parseInput({type:'number',dec:2});
参数 值 类型 说明
type: int||number string 浮点类型或正整数,int类型最大输入12位正整数
dec: 4 int 可选，与浮点捆绑的小数位数，默认4位,type类型为int时可以限制int类型的最大位数

overflow: 字符串截断为... " ".overflow

var str = "字符串阶段为"
str.overflow(3) >>> 字符串...
参数 值 实例
[arguments 1] int 截断字符串".overflow(3) >>>> "截断字...
[arguments 2] string "截断字符串".overflow(3,"---") >>>> "截断字---"
[arguments 3] string "截断字符串".overflow(3,"---","鼠标滑过显示内容") >>>> "截断字---（鼠标滑过后显示内容）"

Format: 日期格式化 (new Date()).Format({})

对Date的扩展，将 Date 转化为指定格式的String 月(M)、日(d)、小时(h)、分(m)、秒(s)、季度(q) 可以用 1-2 个占位符， 年(y)可以用 1-4 个占位符，毫秒(S)只能用 1 个占位符(是 1-3 位的数字) 例子：

(new Date()).Format("yyyy-MM-dd hh:mm:ss.S") ==> 2006-07-02 08:09:04.423

(new Date()).Format("yyyy-M-d h:m:s.S") ==> 2006-7-2 8:9:4.18

扩展esayui--datebox只选择月份: $("xxx").DateBox({})

$("xxx").DateBox({})
参数：同esayui的datebox
附加参数 类型 传值/回调
Select function { Select:function( self,yer,mon ){ self--> 返回$(this) ,yer-->年, mon-->月 } }

扩展esayui--DateSelect 选择年/季度/月: $("xxx").DateSelect({})

$("xxx").DateSelect({ min:2014, max:2015, leve:3, start:1, onChange:function(data){ data为选取后的数据} })
附加参数 类型 传值/回调
min int { min:2013 }
max int { min:2013,max:2014}
leve int { leve:3} 层级数，只显示年leve:1，季度leve:2,月leve:3
start int { start:1 启始层级 1：年 2：季度}
onChange function function(date){ date [String] 点击选取的日期后执行 }
